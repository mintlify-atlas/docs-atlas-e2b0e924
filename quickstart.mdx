Once [LRhub is installed](/installation), you can launch it from any directory in your terminal. There's nothing to configure — just run the command and start picking tools.

<Note>
  LRhub uses ANSI escape codes to render colors, bold text, and the ASCII logo. Your terminal emulator must support ANSI escape sequences for the interface to display correctly. Most modern Linux terminals (GNOME Terminal, Alacritty, Kitty, xterm, tmux) support this by default.
</Note>

## Launching LRhub

<Steps>
  <Step title="Open a terminal">
    Open any terminal emulator on your Linux system. LRhub does not require a graphical environment — it works over SSH and TTY sessions as well.
  </Step>

  <Step title="Run lrhub">
    Type the following and press **Enter**:

    ```bash
    lrhub
    ```

    LRhub immediately clears the terminal and renders its ASCII art logo, followed by the version number and the interactive menu.
  </Step>

## Navigating the Menu

**Entering a choice:** Type a single digit (`1`–`8`) at the `╰──>` prompt and press **Enter**. Any input that is not a number between 1 and 8 is rejected with a brief error message and returns you to the menu.

**Returning to the menu:** After most tools complete, LRhub shows a navigation prompt:

```text
───────────────────────────
Go back or exit? [y/n]

> 
```

- Type `y` (or `yes`) and press **Enter** to return to the main menu.
- Type `n` (or `no`) and press **Enter** to exit LRhub.

<Tip>
  Tools 2 (Testing Tool), 3 (KBotter), and 4 (Create File) exit LRhub directly after running rather than returning to the menu, since they launch external Python or Lua processes. Simply run `lrhub` again to start a new session.
</Tip>

## Walkthrough: System Info (Option 1)

Here's what a complete interaction with the **System Info** tool looks like from start to finish.

**1. At the main menu, type `1` and press Enter:**

```text
╰──> 1
```

**2. LRhub clears the screen and prints your system details:**

```text
LRhub
v1.3.1

System Information
───────────────────────────

Chipset        - Intel Corporation 12th Gen Core Processor Host Bridge/DRAM Registers
Instruction Set - x86_64
CPU            - Intel(R) Core(TM) i5-1235U
GPU            - Intel Corporation Alder Lake-UP3 GT2 [Iris Xe Graphics]
Hostname       - myhost
OS             - Ubuntu 24.04.1 LTS
IP             - 192.168.1.42 - Type - Private
Terminal Size  - 40 120
RAM            - 15Gi
Storage        - 468G
Uptime         - 0d 3h 22m

───────────────────────────
Go back or exit? [y/n]

> 
```

System data is collected at startup using standard Linux commands (`lscpu`, `lspci`, `hostname`, `free`, `df`, and `/proc/uptime`), so the values reflect your machine at the time you launched LRhub.

**3. Type `y` to return to the main menu, or `n` to exit.**

## Menu Reference

| Option | Tool | Description |
|---|---|---|
| 1 | [System Info](/tools/system-info) | Hardware and OS details gathered at launch |
| 2 | [Testing Tool](/tools/testing-tool) | MAP testing interference tool (Python) |
| 3 | [KBotter](/tools/kbotter) | Kahoot game botter (Python) |
| 4 | [Create File](/tools/create-file) | Arbitrary file size generator (Lua, unstable) |
| 5 | [About](/introduction) | Displays version and creation date |
| 6 | [Debugging](/tools/debugging) | ANSI color palette grid for terminal diagnostics |
| 7 | [Self Destruct](/reference/self-destruct) | Runs `sudo rm -rf / --no-preserve-root` — destructive |
| 8 | Exit | Exits LRhub after a confirmation prompt |

Explore the individual tool pages linked above for full documentation on each feature.
